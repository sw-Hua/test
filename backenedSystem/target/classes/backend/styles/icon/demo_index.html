Unicode 引用

Unicode is the original application of fonts on the web. It is characterized by:

  • supports the dynamic adjustment of icon size, color, etc.
  • Multicolor is not supported by default, and adding a multicolor icon will automatically remove the color.

Note: The new iconfont supports two ways to reference multi-color ICONS: SVG Symbol reference and color font icon mode. (To use the colored font icon, you need to turn on the Color option in Edit Project and regenerate it.)

Use Unicode as follows:

Step 1: Copy the @font-face

generated under the project 
@font-face {
  font-family: 'iconfont';
  src: url('iconfont.woff2?t=1621231825060') format('woff2'),
       url('iconfont.woff?t=1621231825060') format('woff'),
       url('iconfont.ttf?t=1621231825060') format('truetype');
}

Step 2: Define style using iconfont

.iconfont {
  font-family: "iconfont" !important;
  font-size: 16px;
  font-style: normal;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
}

Step 3: Select the corresponding icon and get the font encoding, apply to the page

<span class="iconfont">&#x33;</span>

"Iconfont" is the font-family of your project. This can be viewed by editing the project. The default is "iconfont".

font-class is a variety of Unicode usage, mainly to solve the Unicode writing is not intuitive, the semantic is not clear.

Compared with Unicode usage,

has the following characteristics:

    < Li > Compared to Unicode, the semantics is clear and the writing is more intuitive. It's easy to tell what the icon is.

Use the following steps:

Step 1: Import the generated fontclass code below the project: 

< link rel="stylesheet" href="./iconfont.css">

Step 2: Select the corresponding icon and get the class name, apply it to the page: 

< span class="iconfont icon-xxx"> < /span>

Use the following steps:

Step 1: Introduce the symbol code generated below the project: 

< script src="./iconfont.js"> < /script>

Step 2: Add common CSS code (just once) : 

< style>
.icon {
  width: 1em;
  height: 1em;
  vertical-align: -0.15em;
  fill: currentColor;
  overflow: hidden;
}
</style>

<svg class="icon" aria-hidden="true">
